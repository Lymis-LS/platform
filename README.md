# Bimdata Platform
Bimdata Platform application.
## Up & running
1. Install dependencies:
```
npm i
```
2. Configure environment
```
cp .env.example .env
```
3. Change in .env any custom configuration

4. Starting dev server:
```
npm run dev
```
5. compile svg icons:
```
npm run svg-process
```
6. Building for production:
```
npm run build
```
7. Running tests (unit, e2e, all):
```
npm run unit
npm run e2e
npm run test
```
